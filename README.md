## Installation
1. Clone the repository:
```
git clone git@github.com:Rayns15/django-website-tools.git
```
2. Navigate to the directory:
```
cd toolsweb
```
3. Run the server:
```
python manage.py runserver
```
4. Open your web browser and go to http://localhost:8000.
